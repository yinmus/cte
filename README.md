# CTextEditor

**Простой текстовый редактор на C** 

**Команды:**

[Alt + U] - Command line

[Alt + X] - Cut line

**Использование:**
```bash
cte filename
```
___
<details>
  <summary>Команды</summary>
  <p>q - quit</p>
  <p>w - write</p>
  <p>wq - write quit</p>
</details>
